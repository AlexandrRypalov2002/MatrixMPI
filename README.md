# Описание

В репозитории реализовано умножение матриц с помощью MPI. 
Для вывода полученной матрицы в файл необходимо указать в ofstream в main.cpp и main_serial.cpp пути до необходимых файлов 

# Результаты запусков

Рассматривались две матрицы размера 1000*1000 
|N | Time | 
|---:|----|
|1  |3.7|
|2  |2.8|
|5  |2.4|
|10 |1.9|

# Сборка
cmake . -B build/

cmake --build build/

Собирается два исполняемых файла - matrix-mpi и matrix-mpi-serial, предзначенных, соответственно, для параллельного и последовательного запуска умножения матриц
